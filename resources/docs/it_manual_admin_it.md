# IT System - Admin (IT) User Manual

Welcome to the IT Admin system manual. As an **IT Admin**, you have full access control over all resources in the IT system.

## Key Responsibilities

1. **Dashboard Monitoring**:
   - Oversee active IT inventory items, write-offs, and pending e-waste disposal requests.
2. **User Management**:
   - Create, edit, and deactivate IT user accounts.
   - Assign user roles such as HOU, GM, CEO, Finance Admin, and Staff.
3. **Master Data Administration**:
   - Maintain categories, departments, models, and locations for assets.
4. **Audit Logs & Activity Review**:
   - Track all system actions, login activities, modifications, and inventory exports.
