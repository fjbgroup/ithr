# Admin (IT) User Manual

As an **Admin (IT)**, you have full administrative rights over the portal, user accounts, system configuration, system security settings, and system-wide activity logs.

---

## 1. System Administration & Security Control

### 1.1 User Accounts Management
- Go to **System Security** -> **User Accounts**.
- **Create Account**: Create portal credentials for employees. Select a Staff Member, assign a role (Staff, Admin HR, Admin IT, Head of Unit, General Manager, CEO), and set their access level.
- **Manage Roles**: Map user profiles to appropriate modules (HR, IT System, WT System roles).
- **Reset Password**: Reset user passwords when requested.
- **Toggle User Status**: Activate or deactivate accounts. Deactivated users cannot log into the HR portal, IT system, or WT system.

### 1.2 System-wide Security Toggles
At the top of the User Accounts dashboard, you can toggle critical system configurations:
- **Email Sending**: Turn on/off email notifications (e.g. training invitations, update request alerts).
- **TOTP (MFA) Status**: Toggle system-wide requirement for Two-Factor Authentication.

---

## 2. Audit Logs
Monitor system transactions and security compliance.

1. Go to **System Security** -> **Audit Log**.
2. Review records of critical actions:
   - User log-ins/log-outs.
   - Profile update requests and approvals.
   - Staff records creations, updates, or deletions.
   - Settings toggled by IT Admins.
3. Filter by action, date range, or username to investigate specific incidents.

---

## 3. Role Metrics
- Navigate to **Role Matrix** or **Role Metrics** (if available).
- View distribution diagrams and numbers of Staff, Admin (HR), and Admin (IT) roles inside the portal.
- Verify access mappings to ensure strict adherence to security rules.

---

## 4. Integration with IT & WT Systems
The portal acts as a central gateway. You can easily switch to:
- **IT System**: Click **IT System** in the sidebar to jump to the IT Inventory, Assets, and Service Ticketing portal.
- **WT System**: Click **WT System** in the sidebar to switch to the Walkie Talkie Management module.
