# FJB Internal Systems User Manual

This project contains the HR portal, IT Inventory System, and WT System used by FGV Johor Bulkers staff.

## User Manual Location

The user manual is available from the bottom section of the left navbar.

| Account/System | Navbar Section | Link |
| --- | --- | --- |
| WT ICT account | Bottom `Help` section | `ICT User Manual` |
| WT Executive account | Bottom `Help` section | `Executive User Manual` |
| IT Inventory account | Bottom `Help` section | `User Manual` |

For WT accounts, the manual label changes based on the active account view. ICT users see the ICT guide, while Executive users see the Executive guide.

## WT System Quick Guide

1. Login through `/wt`.
2. Use the left navbar to open request, inventory, approval, status, and account pages.
3. Scroll to the bottom `Help` section of the navbar.
4. Open `ICT User Manual` or `Executive User Manual`.
5. Use `Back to Portal`, `IT System`, or `HR Portal` from the same bottom section when switching systems.

## IT Inventory Quick Guide

1. Login to the IT Inventory System.
2. Use the left navbar for dashboard, inventory, requests, reports, admin, and master data.
3. Scroll to the bottom `Help` section.
4. Open `User Manual` for role-specific IT Inventory guidance.
5. Use `Role Metric`, `Back to Portal`, `WT System`, or `HR Portal` from the same bottom utility area.
